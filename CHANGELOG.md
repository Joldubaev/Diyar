## [3.3.0] - 2024-05-29

### Added
- Improved user interface for better user experience
- Enhanced order tracking with real-time updates
- New feature to manage user profiles more effectively
- Additional restaurants and cuisines added to the browsing list

### Changed
- Updated project dependencies to the latest versions
- Refactored code for better performance and maintainability

### Fixed
- Fixed various bugs related to order placement and tracking
- Resolved issues with the user profile management

## [3.3.0] - 2024-01-15

### Added
- Initial release of Diyar app with core features:
  - Browse a wide variety of restaurants and cuisines
  - Order food for delivery with a few taps
  - Track your order status in real-time
  - View order history and manage your profile

# Журнал изменений для версии 3.3.4

## [3.3.4] - 2024-08-29

### Что изменилось:

#### Оптимизация изображений и навигации:
- **Оптимизация изображений**: Улучшена скорость загрузки изображений и сокращено время загрузки приложения.
- **Улучшение навигации**: Переработаны элементы интерфейса для более плавной навигации и быстрого доступа к основным функциям.
- **Улучшение карты**: Повышена точность отслеживания местоположения в реальном времени и отображения маршрутов. Оптимизирован алгоритм обновления координат пользователя.

#### Исправления ошибок:
- **Отображение карты**: Исправлены проблемы с отображением карты в некоторых регионах.
- **Стабильность навигации**: Решены проблемы зависания приложения при переключении между разделами.
- **Обновление местоположения**: Исправлены проблемы с сохранением и обновлением данных о местоположении в фоновом режиме.

## [3.3.5] - 2024-11-01

### Что нового

- Исправлены ошибки в курьерской части приложения, улучшена стабильность и производительность.
- Обновлены и улучшены описания блюд для более точного отображения информации.
- Улучшен интерфейс на страницах с продуктами, добавлены более плавные анимации и управление количеством блюд.
