# Diyar

![Diyar](assets/images/app_logo.png)

Diyar is a food delivery app built with Flutter, providing a seamless and efficient way for users to order food from their favorite restaurants and have it delivered to their doorstep.

## Features

- Browse a wide variety of restaurants and cuisines
- Order food for delivery with a few taps
- Track your order status in real-time
- View order history and manage your profile
